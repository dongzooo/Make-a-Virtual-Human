## 가상 인플루언서 만들어보기(Make-a-Virtual-Human)


### 1. Overview
- 가상 인플루언서와 버츄얼 유튜버들이 큰 인기를 끌면서 가상의 인물은 어떤 기술을 써서 만들어 보는 가에 대한 궁금증에서 시작
- GAN 알고리즘을 활용해 가상인물 제작 및 딥페이크 영상 제작
